# Передача контекста (Claude Code / агент)

Понял: принимаю текущее состояние папки проекта как актуальный эталон сайта для дальнейших правок и советов.

## 1. Критично для агента

- **Эталон** — текущее состояние репозитория (рабочее дерево) считается источником правды для визуала и верстки. Не восстанавливайте старые версии «вслепую».
- **Не делать**
  - `git reset --hard HEAD~N` без явной необходимости и понимания последствий.
  - `git revert` корневого (initial) коммита — это ломает историю непредсказуемо.
- **Перед массовым откатом** — сначала `git stash` **или** отдельная ветка, чтобы можно было вернуться к текущему виду сайта без потерь.
- **Если «пропали» файлы** — смотреть `git reflog`, находить нужный коммит и восстанавливать **выборочно** (checkout/ cherry-pick / reset к известному хорошему коммиту), а не откатывать всю историю через один глобальный `revert`.
